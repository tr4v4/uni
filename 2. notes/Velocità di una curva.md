# Velocità di una curva
---
## Introduzione
> La **velocità di una [[Curva|curva]]** $r$ di [[Classe C|classe]] $C^{1}$ sull'intervallo del dominio ($]a, b[$) è il vettore tangente alla curva data, definito come $$v(t) = r'(t) = (r_{1}'(t), \cdots, r_{n}'(t))$$
